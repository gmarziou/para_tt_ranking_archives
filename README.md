# Para Table Tennis world rankings from 1997 to 2024

This site is an archive of the world rankings
Over time we have used 2 systems:

* [From 1997 to end 2011](1997_2011/index.html): based on number of wins independently from who was the opponent
* [From 2012 to end 2024](2012_2024/index.html): based on ratings of opponents

As of 2025, the new rankings will be published weekly on the [ITTF website](https://www.ittf.com/ittf-world-para-ranking/)
